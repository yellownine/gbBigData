## 2. Вывести с помощью цикла while все четные числа от 0 до 100 включительно.  
Подготовил скрипт и сделал его исполняемым. Содержимое скрипта:  
```bash
#!/bin/bash

a=0

while [ $a -ne 100 ]
do
        a=$(($a+2))
        echo $a
done
```
